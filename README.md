# Bar Reviews

Based off of [Jenn Eng's Rails/React app](https://github.com/jennceng/bar_reviews_react_router).

## Technologies

* Sinatra
* React
* React Router

## Setup

```no-highlight
$ npm install
$ npm start
```

In another terminal...

```no-highlight
$ bundle
$ rails server.rb
```

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).
